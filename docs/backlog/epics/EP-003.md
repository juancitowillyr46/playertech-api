# EP-003 Gestión de Usuarios y Accesos

## Objetivo
Controlar el acceso al sistema.

## Problema que Resuelve
Garantizar que únicamente usuarios autorizados accedan a la información.

## Valor de Negocio
Protección de la información.

## Actores
* Super Admin
* Academic Administrator

## Dominios Involucrados
* User
* Role

## Reglas de Negocio Relacionadas
* Solo usuarios activos pueden autenticarse.
* Cada usuario posee un rol.
* Un usuario pertenece a una academia.

## Flujo de Negocio
1. Crear usuario.
2. Asignar rol.
3. Usuario inicia sesión.
4. Usuario accede según permisos.

## Historias Candidatas
* Iniciar sesión.
* Crear usuario.
* Editar usuario.
* Activar usuario.
* Desactivar usuario.

## Métricas de Éxito
* Usuarios activos.

## Fuera de Alcance
* SSO.
* OAuth.

## Preguntas Abiertas
* ¿Se requerirá MFA?

# Objetivo

Permitir consultar el historial de pagos registrados.

---

# Historia de Usuario

Como administrador de academia

Quiero consultar los pagos

Para revisar el historial de cobros realizados y pendientes.

---

# Reglas de Negocio

* La consulta debe restringirse a la academia actual.
* Debe permitir filtrar por estado cuando aplique.

---

# Criterios de Aceptación

* Dado que existen pagos registrados, cuando los consulto, entonces el sistema los lista correctamente.

---

# Objetivo

Permitir registrar el medio por el que se realizó un pago.

---

# Historia de Usuario

Como administrador de academia

Quiero registrar el medio de pago

Para saber si el cobro fue en efectivo, transferencia o billetera digital.

---

# Reglas de Negocio

* El medio de pago debe ser uno de los valores permitidos.
* El medio de pago debe quedar asociado al pago registrado.

---

# Criterios de Aceptación

* Dado un pago registrado, cuando indico su medio, entonces el sistema lo conserva correctamente.

# Objetivo

Permitir reactivar un equipo previamente desactivado.

---

# Historia de Usuario

Como administrador de academia

Quiero reactivar un equipo

Para volver a utilizarlo dentro de la operación competitiva.

---

# Reglas de Negocio

## BR-001

Solo pueden reactivarse equipos INACTIVE.

---

# Criterios de Aceptación

## CA-001

Dado un equipo inactivo

Cuando se reactiva

Entonces el sistema cambia el estado a ACTIVE.

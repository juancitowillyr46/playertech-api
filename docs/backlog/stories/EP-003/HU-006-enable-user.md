# HU-006 Reactivar Usuario Administrativo

## Historia de Usuario

Como administrador de plataforma o academia

Quiero reactivar un usuario

Para restaurar su acceso a la plataforma.

---

# Reglas de Negocio

* Solo usuarios inactivos pueden reactivarse.

---

# Criterios de Aceptación

## CA-001

Dado un usuario inactivo

Cuando lo reactivo

Entonces el sistema cambia su estado a ACTIVE.

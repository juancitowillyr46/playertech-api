# HU-003 Registrar usuario como miembro del staff

## Epic
EP-021 Gestión de Staff y Cuerpo Técnico

## Prioridad
Alta

## MVP
Sí

## Estado
Draft

## Actor Principal
Academy Admin

## Objetivo
Marcar un usuario activo como miembro del staff de la academia.

## Reglas de Negocio
* El usuario debe estar activo.
* El usuario debe pertenecer a la academia actual.

## Criterios de Aceptación
* Dado un usuario administrativo activo, cuando se registra como staff, entonces queda disponible para asignaciones técnicas.

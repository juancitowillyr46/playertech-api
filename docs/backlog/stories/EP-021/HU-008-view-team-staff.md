# HU-008 Consultar cuerpo técnico de un equipo

EP-021 Gestión de Staff y Cuerpo Técnico

## Prioridad
Media

## MVP
Sí

Academy Admin

## Objetivo
Consultar el cuerpo técnico activo de un equipo.

## Reglas de Negocio
* Debe respetar el `academy_id`.
* Debe mostrar roles técnicos vigentes.

## Criterios de Aceptación
* Dado un equipo con staff asignado, cuando se consulta, entonces se listan sus miembros y funciones.

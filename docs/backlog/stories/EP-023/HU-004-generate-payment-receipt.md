# Objetivo

Permitir generar un comprobante operativo asociado a un pago.

---

# Historia de Usuario

Como administrador de academia

Quiero generar un comprobante de pago

Para entregar un soporte descargable del recaudo realizado.

---

# Reglas de Negocio

* El comprobante debe estar asociado a un pago existente.
* El comprobante operativo no reemplaza una factura fiscal.
* El comprobante debe poder descargarse o visualizarse.

---

# Criterios de Aceptación

* Dado un pago válido, cuando genero el comprobante, entonces el sistema lo crea correctamente.
